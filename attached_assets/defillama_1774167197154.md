# DefiLlama API Reference

## Overview

DefiLlama is the largest DeFi TVL aggregator. Fully free, no API key, no auth.
Multiple sub-APIs live on different subdomains.

---

## Core API — `https://api.llama.fi`

### TVL

| Endpoint | Description | Key Params |
|---|---|---|
| `GET /v2/protocols` | All protocols with current TVL, chain breakdown, token info | None |
| `GET /protocol/{name}` | Single protocol — historical TVL, chain breakdown, token TVL | `name` = slug (e.g., `aave`, `lido`) |
| `GET /v2/chains` | All chains with current TVL | None |
| `GET /v2/historicalChainTvl` | Global TVL history | None |
| `GET /v2/historicalChainTvl/{chain}` | Single chain TVL history | `chain` = chain name |

**Gotchas:**
- Protocol slugs are lowercase, hyphenated. Check `/v2/protocols` to find exact slug.
- `/protocol/{name}` returns daily granularity. `tvl` array has `{date, totalLiquidityUSD}`.
- `chainTvls` object in protocol response breaks down by chain + includes borrowed/staking if relevant.
- Some protocols show `tvl: 0` if they've deprecated or migrated (e.g., old versions).

### Fees & Revenue

| Endpoint | Description |
|---|---|
| `GET /overview/fees` | All protocols — 24h fees, revenue, and historical |
| `GET /overview/fees/{chain}` | Fees by chain |
| `GET /summary/fees/{protocol}` | Single protocol fee history |
| `GET /overview/revenue` | Revenue specifically (portion of fees going to protocol/token holders) |

**Response shape** (summary/fees):
```json
{
  "name": "aave",
  "total24h": 1234567,
  "total7d": 8765432,
  "totalAllTime": 999999999,
  "totalDataChart": [[timestamp, dailyFees], ...],
  "totalDataChartBreakdown": { "v3": [[ts, val], ...] }
}
```

**Gotchas:**
- `fees` ≠ `revenue`. Fees = total paid by users. Revenue = portion accruing to protocol/token.
- Not all protocols have fee data. Check `/overview/fees` first.
- `totalDataChart` timestamps are Unix seconds, not milliseconds.

## Yields API — `https://yields.llama.fi`

| Endpoint | Description |
|---|---|
| `GET /pools` | All pools — APY, TVL, tokens, chain, project | 
| `GET /chart/{pool}` | Historical APY for a single pool |

**Response shape** (`/pools` item):
```json
{
  "pool": "747c1d2a-...",
  "chain": "Ethereum",
  "project": "aave-v3",
  "symbol": "USDC",
  "tvlUsd": 1234567890,
  "apy": 3.45,
  "apyBase": 2.1,
  "apyReward": 1.35,
  "rewardTokens": ["0x..."],
  "underlyingTokens": ["0x..."]
}
```

**Gotchas:**
- Pool IDs are UUIDs, not protocol names. Search by `project` or `symbol` field.
- `apy` = `apyBase` + `apyReward`. Base = organic yield, Reward = token incentives.
- ~10,000+ pools returned. Filter client-side by chain/project/tvl threshold.
- Some pools show `apy: null` — skip these.

## Prices API — `https://coins.llama.fi`

| Endpoint | Description |
|---|---|
| `GET /prices/current/{coins}` | Current price by contract address |
| `GET /prices/historical/{timestamp}/{coins}` | Price at specific timestamp |
| `GET /chart/{coins}` | Price history chart data |
| `GET /percentage/{coins}` | Price change percentages |

**Address format**: `{chain}:{address}` — e.g., `ethereum:0xdac17f958d2ee523a2206206994597c13d831ec7` for USDT.
Also supports: `coingecko:{id}` for CoinGecko IDs.

**Gotchas:**
- Can batch multiple coins in one request: comma-separated in the URL.
- Chain names: `ethereum`, `bsc`, `polygon`, `arbitrum`, `optimism`, `avalanche`, `solana`, `base`, etc.
- Returns `{coins: { "ethereum:0x...": { price, symbol, timestamp, confidence } }}`.
- `confidence` < 0.9 means the price might be unreliable (low liquidity).

## Common Patterns

### Get protocol fundamentals for a valuation model
```
1. GET /protocol/{slug}                    → TVL history
2. GET /summary/fees/{slug}                → Fee & revenue history  
3. GET https://coins.llama.fi/prices/current/coingecko:{id}  → Current price
```

### Compare DEX market share
```
1. GET /overview/dexs                      → All DEX volumes
2. Filter/sort by total24h
3. GET /summary/dexs/{protocol}            → Deep dive on top ones
```

### Track stablecoin migration
```
1. GET /stablecoins                        → Current breakdown
2. GET /stablecoincharts/{chain}           → Historical for target chains
```

## Rate Limits & Best Practices

- No auth required, but be respectful — batch where possible.
- If you get 429s, back off 5s and retry. Happens rarely.
- Cache `/v2/protocols` response — it's large (~2MB) and changes slowly.
- For bulk price lookups, use `coins.llama.fi/prices/current/{comma,separated,list}` instead of individual calls.
- Timestamps in responses are Unix seconds unless noted otherwise.
