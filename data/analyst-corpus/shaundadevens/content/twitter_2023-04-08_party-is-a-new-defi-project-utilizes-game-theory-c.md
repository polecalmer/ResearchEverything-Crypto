In simple terms the expected value of Pool A and Pool B are always equal. Since its a 0 sum game we expect a 0% gain on both pools.

To better understand we can calculate Returns, Win %, and EV for each Pool.

*We will use Pool A as an example, for Pool b its the same.

Party has 2 main Pools - Pool A &amp; Pool B

Unlike traditional pools there are no $party emissions; The winning pool gets 10% of the tokens from the losing pool. This means that the game is 0 sum. Rewards are calculated with risk-neutral strategy based martingale systems. https://t.co/IrTUtUjcWL

2. Winning %

The winning chance considers the ratio of tokens staked. Pools with higher tokens have higher chances of winning. 

Win % = Pool A / Pool A + Pool B

If Pool A has 1m $party tokens and Pool B has 2m; the chance of winning is 33% for Pool A

1. Winning Returns 

The winning pool gets 10% of the other pools tokens; so the for Pool A is 10% of Pool Bs tokens.

Returns = (10% * Pool B) / Pool A

If Pool A has 1m $Party tokens and Pool B has $2m; if  Pool A wins we get 200k $party Distributed - or 20% Returns.

3. Expected Value:

However, If we lose we forfeit 10% of our tokens. We compute EV calculating each outcome by its Probability

Outcome 1: Winning
-&gt;Win Return * Winning Chance
Outcome 2: Losing
-&gt;Loss (Always 10%) * Losing Chance

In this example our EV:
.2*.33+(-.1)*.66 = 0%

The fun and strategy starts with $FAVOR

Favor is a mechanism that provides a bonus to the win rate of a Pool based on the ratio of one Pool's Favor to the other.
